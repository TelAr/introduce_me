    코드로 보여주는 준비된 개발자
    중앙대학교 컴퓨터공학부 소프트웨어 전공 4년 재학을 한 예비 개발자 박호연


    어릴 때부터 게임을 좋아했고, 좀 더 확대되어 입력에 따른 출력을 생성해내는 프로그래밍에 관심을 가졌음, 프로그램을 보는 시각을 게임으로 대입하고 어떤 구조로 작동해야 할 지 분석을 시도해보며, 내가 원하는 게임을 종종 간단하게 개발을 시도해보기도 하였음
    한번 개발 목표로 삼은 프로그램에 대해, 특정 기능을 개발하고자 하면 정석 방법을 우회해서라도 구현을 최우선으로 하고자 함, 한번 기능이 안정화되면 그 이후로는 자원과 시간을 덜 쓰는 방법으로 개선하고자 함
    대학교 재학중에도 C나 C++, 그 중에서도 포인터와 같은 내부 메모리에 접근하는 형태의 프로그래밍에 관심이 많았고 개발 방향성도 그 위주로 진행하였음, 물론 Python과 같은 스크립트 언어에 대해서도 관심을 가지고 개발 역량을 쌓아오고 있음
    올해 들어와서 TCP, UDP 통신 프로토콜에 대한 지식을 쌓기 시작하였고, 이를 이용한 과제 수행도 원활하게 진행한 경력이 있음

    프로젝트 및 진행한 일들

    1. 학교 과 동아리 중 당시 게임개발 동아리였던 스티머즈 회장직 (2017.03~2018.02)

    • 회장 직 기간동안 게임 개발 대회 진행 및 동아리 내 게임 개발 관련 프로젝트(예: GIMP를 활용한 이미지 리소스 생성 방법, 유니티 엔진을 통한 게임 개발 방법 등) 진행힘

    2. Java를 이용한 자동 필기 프로그램 팀 프로젝트

    • 해당 프로젝트 내부에서 이미지 자료 대조 및 유사도 측정 부분 작업 진행
    • 영상 내 프레임별로 추출된 이미지 자료를 대조하여 필기 추가 여부, 페이지 전환 여부를 판독하여 프로그램 구동을 할 수 있게 함
    • 개발 당시 인공지능에 대한 지식과 네트워크에 대한 지식이 부족하였기에 관련 기능이 구현이 미흡함, 그러나 파라미터 입력이 가능하였기에, 추후 개발 재시도시 기능 추가 구현 가능함
    • // https://github.com/TelAr/Lecture-Assistant

    3. Python을 활용한 TCP, UDP 기반의 통신 채널 확립 과제 수행 경력

    • 기초적인 수준의 프로그램이나, 채팅 어플리케이션으로서 요구되는 기본 기능 및 통신 기능 구현 성공
    • TCP의 경우 채팅 헤더에 따른 다양한 기능을 수행하면서 안정적인 서버-클라이언트 교신 환경 구축 성공
    • UDP의 경우 P2P 교신 채널을 활성화시키면서 최소한의 서버 개입 환경으로도 교신이 원활할 수 있는 환경 구축에 성공함
    • // https://github.com/TelAr/NetworkApplicationAssignment

    4. OpenCV를 활용하여 행동을 추적하여 시스템 내 입력을 주는 프로그램 제작 프로젝트

    • 해당 프로젝트 내에서 입력 받는 영상 자료에서 단색 오브젝트를 감지하고 필터링 하는 역할 수행
    • 리얼타임으로 구동되는 프로그램에서 연산 과부하를 방지하기 위해 픽셀을 몽쳐서 연산하는 방식을 통해 리얼타임에서도 정상작동 되도록 프로그램을 개선함
    • // https://github.com/heebinYoo/IP_proj2 해당 프로젝트에서 TelAr 계정으로 Contribute함

    5. 유니티 엔진을 이용한 개인 개발 게임

    • 초보적인 수준의 기능 구현이기는 하나, 엔진 내 렌더링, 오브젝트 간 상호작용, 파일 입출력, 씬 전환과 같은 부분에서의 기능은 구현 완료 상태
    • 추가로 구현할 기능으로는 TCP 통신을 통한 서버-클라이언트 교신 구축, 게임 내 기록 신뢰성 및 통신 신뢰성 구축 등 통신 관련 기능 및 신뢰성 기능을 구현하고자 함
    • 비록 게임 환경이기는 하지만, 교신 구현을 통해 백엔드 / 프론트엔드 환경을 구축하여 서버 교신을 시도할 수 있음
    • // https://drive.google.com/file/d/1_-O-NcQzytb2P9ZbpR9fby5X7kZqIN90/view

    활용 가능한 기술

    • C, C++, Java는 어느정도 활용 가능
    • JavaScript, Python, C#은 주어진 코드를 보고 어떻게 활용해야 할 지 작성은 가능한 정도
    • TCP, UDP 통신에 대해 기본적인 지식이 있으며, 이를 활용하여 어플리케이션 내 자체 통신 규약을 새로 확립 후 교신할 수 있는 구조를 만들 수 있음
    • 유니티 엔진에 대해서 어느정도 활용 가능
